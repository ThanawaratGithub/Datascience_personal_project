    โดยนำข้อมูลผู้มาขอกู้เงินจาก kaggle เพื่อใช้ในการทำนายว่าผู้ขอกู้เงินคนไหน มีโอกาสผิดชำระหนี้โดยดูจากประวัติทางการเงิน
